# لوحة تحكم مدينة جبلة

لوحة تحكم كاملة لإدارة محتوى مدينة جبلة الأثرية، مبنية باستخدام React و Firebase.

## المميزات
- إدارة الأخبار والمقالات.
- إدارة المعالم التاريخية.
- إدارة الجدول الزمني التاريخي.
- نظام دخول آمن باستخدام Firebase Auth.
- قاعدة بيانات Firestore.
- دعم الوضع الليلي (Dark Mode).

## المجلدات
- `عام`: يحتوي على الملفات العامة (إذا كان هناك صور أو أيقونات ثابتة).
- `المصدر`: يحتوي على شيفرة المصدر الرئيسية (React components).

## التشغيل
1. قم بتثبيت التبعيات: `npm install`
2. ابدأ التشغيل في وضع التطوير: `npm run dev`
3. للبناء النهائي: `npm run build`
